Especificações Técnicas

Escopo do Projeto

A IA Wex será utilizada em múltiplos canais e dispositivos, incluindo:

  • WhatsApp: Suporte direto ao usuário.
  • Web: Interface para acompanhamento de estatísticas e feedback.

Camadas da Interface

  • Entrada: Usuário envia mensagens suspeitas via WhatsApp ou Web.
  • Processamento: NLP e Machine Learning analisam a mensagem e calculam um score de risco.
  • Saída: A IA fornece diagnóstico e recomendações ao usuário.

Arquitetura e Tecnologias

  • Backend: Node.js (Express), DynamoDB (AWS), AWS Lambda.
  • Integração: API do WhatsApp Business (Meta).
  • Machine Learning: TensorFlow.js para análise de padrões.

Fluxo de Processo

  • Usuário envia mensagem suspeita via WhatsApp ou Web.
  • A IA analisa o texto e compara com base de golpes conhecidos.
  • A IA responde com diagnóstico e sugestões.
  • Usuário fornece feedback para melhoria contínua.

Integração com WhatsApp

  • API: Configuração via Facebook Developer.
  • Mensagens: Uso de Webhooks para recebimento.
  • Respostas: Geradas automaticamente pela IA.
  • Segurança: Logs anonimizados e armazenados no DynamoDB.
